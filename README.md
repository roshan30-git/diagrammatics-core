# Diagrammatics Core

**A local-first knowledge and reasoning engine that structures information into connected systems for better understanding, recall, and study.**

## 🧠 Core Philosophy

Diagrammatics Core is designed for students, teachers, and engineers who need a premium, locally-hosted academic command center. It leverages multi-agent orchestration to transform complex information into structured, visual knowledge.

## 🚀 Key Features

- **Visual Synthesis Engine (VSE)**: Renders complex STEM diagrams (Circuits, Flowcharts, Math) directly.
- **Multi-Agent Swarm**: Specialized agents for theory, diagrams, and verification working in parallel.
- **Knowledge Memory Palace**: Hierarchical RAG system indexed by Semester, Subject, and Topic.
- **100% Local-First**: Your data never leaves your machine. Powered by `sqlite-vec`.

## 🛠️ Tech Stack

- **Frontend**: Vanilla HTML5, CSS3 (Glassmorphism), JavaScript
- **Icons**: Lucide
- **Typography**: Inter & JetBrains Mono

## 📂 Project Structure

- `index.html`: Main dashboard entry point.
- `style.css`: Premium design system and glassmorphic UI.
- `main.js`: Interactive logic and agent simulations.
